# 💍 Will You Marry Me? – Interactive Proposal Website

A fun and romantic interactive proposal webpage built with **HTML, CSS, and JavaScript**.

The page displays a beautiful proposal card asking:

> **"Will you marry me?" 💖**

The **"Yes"** button celebrates the moment with a romantic success message and confetti, while the **"No"** button playfully escapes whenever someone tries to click it.

---

## ✨ Features

* 💍 Romantic proposal popup
* ❤️ Animated floating hearts
* 🎉 Confetti celebration after clicking **Yes**
* 😄 "No" button that jumps to random locations
* ✨ Sparkle animation whenever the button moves
* 📱 Fully responsive for desktop and mobile devices
* 🎨 Modern glassmorphism design
* 🚀 No external libraries required

## 🎯 How It Works

* The page opens with a romantic proposal card.
* The **Yes** button accepts the proposal and displays:

  * A celebration message
  * Falling confetti
  * Romantic animations
* The **No** button:

  * Detects when the cursor or finger gets close.
  * Instantly moves to another location.
  * Displays playful messages.
  * Leaves behind a sparkle animation.
